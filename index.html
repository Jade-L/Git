Commande Git

Commande Action
Interaction
gti status Liste les fichiers modifiés/nouveaux
git diff Liste les modifications
git commit Enregistre le travail effectuer
git add Place les modifications en attente de validation
git rm Supprime le fichier du répertoire
git mv Rennome le fichier
Commande Action
Les Branches
gti branch Liste les branches existante
git branch + nom de la branche Créer une nouvelle branch
git checkout + nom de la branche Bascule sur la branche spécifié
git merge + nom de la branche Combiner dans la branche courante l'historique de la branche spécifiée
git branch -s Supprime la branch
