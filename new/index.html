Commande Git

Interaction

  • git status Liste les fichiers modifiés/nouveaux
  • git diff Liste les midifications
  • git commit Enregistre le travail effectué
  • git add Place les modifications en attente de validation/ajoute
  • git rm Suppprime le fichier du répertoire
  • git mv Rennome le fichier

    • Les branches

  • git branch Liste les branches existante
  • git branch + "nom" Créer une nouvelle branche
  • git checkout Bascule sur la branch spécifié
  • git merge Combiner dans la branche courante l'histoire de la branche spécifié
  • git branch -s Supprime la branche
